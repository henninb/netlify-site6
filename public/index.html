Connect Napster

Authenticate with your Napster account to stream your favorites.

This is a demonstration of using OAuth to authenticate with Napster.
This login flow uses PKCE (Proof Key for Code Exchange),
so the website doesn't have to expose the client secret in the source code.

How it works

1
First, the website generates a random alphanumeric string of length 64 — the Code Verifier. 

          

        

      


      

        

          2
          

            You will then be redirected to the login page, where you confirm
            the requested permissions. The URL contains a hash of the Code Verifier
            — the Code Challenge.


            Only this website knows the actual Code Verifier. This will be useful later on.
          

        

      


      

        

          3
          

            After granting permission, you are redirected back here with an
            auth code. The website exchanges this for an access token
            that allows it to read your Napster data — and proves its identity to the API
            by supplying the unhashed Code Verifier.
